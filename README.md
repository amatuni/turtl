# turtl

[![Build Status (Travis)](https://travis-ci.org/andreiamatuni/turtl.svg?branch=master)](https://travis-ci.org/amatuni/turtl)


A p2p register based virtual machine.


### install

```
$ git clone https://github.com/andreiamatuni/turtl.git
$ cd turtl
$ cargo install
```

### start

```
$ turtl shell
```
