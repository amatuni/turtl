**Note: this is a work in progress**

<br>

### install

```bash
$: go get -d github.com/andreiamatuni/turtl
$: cd $GOPATH/src/github.com/andreiamatuni/turtl
$: make install
```

### run
Start the turtl shell:

```bash
$: turtl shell
```
